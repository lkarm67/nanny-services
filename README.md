# Nanny Services

## 📌 About the Project

**Nanny Services** is a web application for finding and booking babysitters.

The application allows users to browse a list of available nannies, filter them by different criteria, add them to favorites, and schedule an appointment.

The project is built as a **Single Page Application (SPA)** using modern frontend technologies.

### Main Features

* Browse a list of nannies
* Filter and sort nannies
* Add nannies to favorites
* User authentication
* Schedule an appointment with a nanny

---

## 🚀 Technologies Used

* React
* TypeScript
* Vite
* React Router DOM
* Firebase (Authentication & Database)
* React Hook Form
* Yup (form validation)
* React Hot Toast (notifications)
* CSS Modules

---

## 📂 Project Features

### 🔎 Nannies List

* loading nanny data from JSON / Firebase
* displaying nanny cards
* pagination with a **Load More** button

### 🎛 Filters

Users can filter nannies by:

* price
* rating
* alphabetical order

### ❤️ Favorites

* add nannies to favorites
* view saved nannies on the Favorites page
* favorites are stored for the authorized user

### 🔐 Authentication

* user registration
* login
* logout
* UI updates after authentication

### 📅 Appointment

Users can schedule an appointment with a nanny through a modal form.

## ⚙ Installation and Running the Project

### 1. Clone the repository

```
git clone https://github.com/lkarm67/nanny-services.git
```

### 2. Navigate to the project folder

```
cd nanny-services
```

### 3. Install dependencies

```
npm install
```

### 4. Run the project

```
npm run dev
```
